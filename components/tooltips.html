--- layout: page title: Tooltips ---

These tooltips are like bubbles but are a pure CSS solution. Hover over the below version badge.

v{{ site.version }} 
<span class="c-badge c-tooltip c-tooltip--right" aria-label="This is the current version of Blaze">v{{ site.version }}</span>

Add c-tooltip and --top, --left, --bottom or --right modifiers to position the tooltip.

Use the aira-label attribute for the text of the tooltip.
